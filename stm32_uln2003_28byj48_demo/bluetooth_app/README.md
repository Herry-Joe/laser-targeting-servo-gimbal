# 蓝牙上位机控制面板 (JDY-31 SPP)

## 📱 快速开始

### 电脑（推荐）
JDY-31 是 **SPP 经典蓝牙**，电脑推荐用 **Web Serial** 控制面板：

1. 电脑蓝牙配对 JDY-31（密码 `1234`）
2. 设备管理器→端口→记下蓝牙 COM 口号
3. 打开 `webserial.html`（需 http 服务）：
   ```bash
   python -m http.server 8000
   ```
   浏览器打开 `http://localhost:8000/webserial.html`
4. 点「🔗 连接」→ 选蓝牙 COM 口

### 手机
安卓装 *Serial Bluetooth Terminal*，连 JDY-31 后直接发命令。

> ⚠️ iOS 无 SPP 串口 App，需用电脑。

## 🎮 功能特性 (webserial.html)

| 功能 | 操作 |
|------|------|
| **方向控制** | ↑↓ 俯仰 ±5° · ←→ 水平转停 · Space 停止 |
| **键盘快捷键** | 方向键 / WASD / Space / 数字 1-9 调速度 / P 选水平 / T 选俯仰 |
| **自定义命令** | 底部文本框输入任意命令（如 `+200`、`r45`） |
| **# OK 高亮** | 串口回显中 `# OK` 绿色、`# ERR` 红色、发送命令青色 |
| **速度滑条** | 1300~10000µs 实时调节 + 自动发送 `vN` 命令 |
| **轴状态指示** | 当前选中轴高亮显示（Pan / Tilt） |
| **预设动作** | 转 90°/45°/10°、+100/-100 步、扫描、跳舞 |

## 📡 通信协议

波特率 **9600**（JDY-31 默认），命令以 `\n` 结尾（点按钮自动带）

| 命令 | 功能 | 示例 |
|------|------|------|
| `p` / `t` | 选 Pan / Tilt 电机 | `p` |
| `c` / `a` | 连续正转 / 反转 | `c` |
| `s` | 停止 | `s` |
| `f` / `h` | 全步 / 半步 | `f` |
| `rN` | 旋转 N° | `r90` |
| `wN` | 扫描 ±N°/2 | `w180` |
| `vN` | 速度 N µs/步 | `v3000` |
| `+N` / `-N` | 前进 / 后退 N 步 | `+200` |
| `d` | 跳舞！ | `d` |
| `?` | 查询状态 | `?` |

STM32 端 USART2（PA2/PA3）接 JDY-31，USART1（PA9/PA10）接 USB-TTL（115200bps），两路均支持上述命令。
